# Auto_Wechat_Replier_Based_On_QW
a very simple but interesting project, auto replying without access towards itchat/wxautox 
wards itchat/wxautox 

这是一个刚开学的无聊大学生做的一个技术力低下的项目。分享出来主要是觉得好玩。
整体内容就是一个基于通义千问的微信自动回复程序（不需要授权，用的是uiautomation）。不过回答的都比较傻（因为我用的是免费的大模型）

※所有的东西都要自己配置哦

※要自己填一个千问的api_key和一个刷新用的朋友（比如'文件传输助手'）
